Start SBT
---------

```bash
$ cd gatling-sbt-plugin-demo
$ sbt
```

Compile the simulations
-----------------------

```bash
> compile
```

Run all simulations
-------------------

```bash
> test
```

Run a single simulation
-----------------------

```bash
> testOnly spicinemas.simulations.EndToEndSimulation
```

Open the last report
--------------------

```bash
> lastReport
```

Data
--------------------
```bash
> sh load_sessions.sh
> sh load_users.sh
````
